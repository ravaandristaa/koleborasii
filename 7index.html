Misal :

x = Harga 1 Ecoprint mug

y = Harga 1 Lampu Hias

z = Harga 1 Jam Kertas

Model Matematika :

  1. 3x + y + z = 158.000
  2. 2x + y + 2z = 170.000
  3. 2x + 3y + 2z = 250.000

Penyelesaian:

Eliminasi pers 1 dan 2

3x + y + z = 158.000

2x + y + 2z = 170.000

----------------------------

y + 4z = 200.000 (pers. 4)

Eliminasi pers 2 dan 3

2x + y + 2z = 170.000

2x + 3y + 2z = 250.000

2y = 80.000

y = 40.000

Substitusi y = 40.000 ke pers 4

y + 4z = 200.000

40.000 + 4z = 200.000

4z = 160.000

z = 40.000

Substitusi y = 40.000 dan z = 40.000 ke pers 1

3x + 40.000 + 40.000 = 158.000

3x + 80.000 = 158.000

3x = 78.000

x = 26.000

Jadi, kesimpulannya adalah :

  1. Harga 1 Ecoprint mug: 25.000
  2. Harga 1 Lampu Hias: 40.000
  3. Harga 1 Jam Kertas: 40.000
