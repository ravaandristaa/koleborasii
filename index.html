SPLTV


Misal :

x = Harga 1 Lampu Hias dari Cup

y = Harga 1 Tempat Flashdisk

z = Harga 1 Tas Laptop

Model Matematika :

  1. 3x + y + z = 130.000
  2. x + 3y + z = 130.000
  3. 2x + y + 2z = 135.000

Penyelesaian:

Eliminasi pers 1. dan 3.

p : 3x + y + z = 130.000 | *2 | 6x + 2y + 2z = 260.000

2x + y + 2z = 135.000 | *1 | x + y + 2z = 135.000

4x + y = 125.000 (pers. 4)

Eliminasi pers 1. dan 2.

p : 3x + y + z = 130.000

x + 3y + z = 130.000

2x - 2y = 0 (pers. 5)

Eliminasi pers 4. dan 5.

4x + y = 125.000 | *2 | 8x + 2y = 250.000

2x - 2y = 0 | *4 | 8x - 8y = 0

10y = 250.000

y = 250.000 : 10

y = 25.000

Substitusi y = 25.000 ke pers 4.

4x + y = 125.000

4x + 25 = 125.000

4x = 125 - 25

4x = 100.000

x = 25.000

Substitusi x = 25.000 dan y = 25.000 ke pers 2.

3x + y + z = 130.000

25(3) + 25 + z = 130.000

75 + 25 + z = 130.000

100.000 + z = 130.000

z = 130.000 - 100.000

z = 30.000

Jadi, kesimpulanya adalah :

  1. Harga 1 Lampu Hias dari Cup = 25.000
  2. Harga 1 Tempat Flashdisk = 25.000
  3. Harga 1 Tas Laptop = 30.000
