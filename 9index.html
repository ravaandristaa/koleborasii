Paket:

Rak Kaleng: 50.000|x

jam kertas: 40.000|y

Ecoprint mug: 25.000|z

Model Matematika :

  1. x + y + z = 115.000
  2. x + 2y + 2z = 180.000
  3. 2x + 2y + z = 205.000

Penyelesaian:

Eliminasi pers 1 dan 2

elim pers (1) dan (2) (x + 2y + 2z) - (x + y + z) = 180.000 - 115.000 y + z = 65.000 (4)

Eliminasi per 3 dan pers 1

2(x + y + z) = 2(115.000) 2x + 2y + 2z = 230.000 (5)

Kurangkan per 3 dan per 5

(2x + 2y + 2z) - (2x + 2y + z) = 230.000 - 205.000 = 25.000

Subsitusi nilai z ke pers 4

y + z &= 65.000 + 25.000 = 65.000 = 65.000 - 25.000 = 40.000

Subsitusi nilai y dan z ke pers 1

x + y + z = 115.000 + 40.000 + 25.000 = 115.000 x 65.000 = 115.000 = 115.000 - 65.000 = 50.000

Jadi kesimpulannya:

  1. Rak Kaleng: 50.000
  2. Jam Kertas: 40.000
  3. Ecoprint mug : 25.000
